typesetting vs desktop publishing, HTML and CSS

typesetting

The arrangement of typesetting has manual placing of letters and symbols on metal plates. This needs focus because every piece of type has to be positioned properly in order to print into words and sentences.

desktop publishing

This is an important part in translation process, more so when considering big scale projects.

HTML and css

HTML does the layout work after that CSS does the presentation

Distinguish between Typesetting as a print technique, and desktop publishing or web technologies. as is the case with HTML and CSS provide more flexibility and performance. Desktop publishing allows for quick and easy change and the integration of multimedia while HTML and CSS offer an experience that can hardly be provided by typesetting.
